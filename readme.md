This is an attempt to revive the old game "Top Speed" with more features and modern code.



Top Speed was originally created by Playing in the Dark. The last known update to the project dates back to approximately 2013, which means the project is now nearly inactive.



I am one of the people who always appreciated the concept behind this game. Given the limited number of audio-based racing games available in our community, I decided to attempt to revive the project using modern gameplay systems and up-to-date code.



Fortunately, I was able to locate the original GitHub repository and discovered that the project is open source. I forked the repository and began rewriting a significant portion of the project in C#.



For reference, the original codebase was written in C++, and the solution was designed to be built using Visual Studio 2008. In addition, it relied on several outdated libraries that are now very difficult to obtain or maintain.



I have been actively working on this project for more than two months. Initially, the goal was to recreate the original game with minimal changes to its core gameplay systems. However, after seeing how many people showed interest in the idea, I decided to expand the scope of the project.



At this point, nearly all game systems have been rewritten from scratch. As a result, the project is no longer intended to be a direct port of the original game, but rather a fully reimagined and more realistic experience with properly designed systems.



The project is still in a very early alpha stage and is not yet ready for a public release. That said, prebuilt versions are automatically generated for every commit on GitHub. These builds are intended strictly for testing and evaluation purposes and may break at any time. They are used to test the game under different conditions and environments.



It is also worth mentioning that while the original game focused solely on racing, I plan to expand the project further by introducing additional content and features.
